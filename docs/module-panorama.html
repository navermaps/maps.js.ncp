Module: panorama

파노라마 서브 모듈은 NAVER 지도 API v3에서 제공하는 서브 모듈 중 하나로, 지정한 DOM 요소에 파노라마 이미지를 표시할 수 있는 뷰어를 삽입하여 거리뷰, 항공뷰의 서비스(이하 파노라마)를 제공할 수 있도록 합니다.
